# Tech Stack 


- Frontend: NextJS
- Backend: FastAPI
- Scraping: Apify for Indeed/LinkedIn + Official API if exist  
- AI layer: 
    - Embedding model: text-embedding-3-small
    - LLM: GPT-4o-mini
- Database: PostgreSQL + pgvector
- API: REST
- Validation: Pydantic
- Migrations: Alembic + SQLAlchemy
- Email: Postmark
- Auth: JWT + fastapi-users
- Queue: ARQ
- Cache: Redis
- CV storage: S3
- Containerization: Docker
